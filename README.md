# Simple listview in Xamarin.Forms

This example explains how to implement the custom control in Xamarin.Forms. In this example, custom control creation is explained using simple listview control.

## Requirements to run the sample

* [Visual Studio 2017](https://visualstudio.microsoft.com/downloads/) or [Visual Studio for Mac](https://visualstudio.microsoft.com/vs/mac/)
* Xamarin add-ons for Visual Studio (available in Visual Studio installer)

## How to run the sample

1. Clone the sample and open it in Visual Studio.

   *Note: If you download the sample using the "Download ZIP" option, right-click it, select Properties, and then select Unblock.*
2. Set any one of the platform specific projects (iOS, Android or UWP) as a start-up project.
3. Clean and build the application.
4. Run the application.
